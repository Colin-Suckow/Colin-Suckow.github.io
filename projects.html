Zagmap Server

Zagmap server is the backend that provides data for zagmap, a data mapping application. The server is built using express and it pulls data from a postgres sql database. The server responds to http requests containg a date with the data on the bike rides taken on that day.

Wattlogix Companion App

This android app allows the user to view information about the battery controller in their rv equiped with a WattLogix battery control module. The app uses Bluetooth LE to retrieve the voltage data from the battery control board and then displays it in the app's interface. This app was written in java and makes use of android's bluetooth library and rxJava for controlling connetion state.
